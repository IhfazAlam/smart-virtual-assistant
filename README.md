# 🤖 Smart Virtual Assistant

A modern Python-based voice-controlled virtual assistant with a beautiful GUI, speech recognition, and automation features.

![Project Screenshot](https://github.com/IhfazAlam/smart-virtual-assistant/blob/a5d79673a83a030f2f770e20bd1084c67c11f405/image.png)

## ✨ Features

- 🎤 Voice Recognition (Speech to Text)
- 🔊 Text-to-Speech Response
- 🌦 Live Weather Updates (OpenWeather API)
- 🌐 Open Google & YouTube instantly
- 🎵 Play Online & Local Music
- 🖥 Modern Tkinter GUI (Dark Theme)
- 🧠 Smart command handling
- ⚡ Fast and lightweight Python project

---

## 🚀 Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/IhfazAlam/smart-virtual-assistant.git
```

### 2️⃣ Open project folder

```bash
cd smart-virtual-assistant
```

### 3️⃣ Create virtual environment (recommended)

```bash
python -m venv venv
venv\Scripts\activate
```

### 4️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run Project

```bash
python gui.py
```
